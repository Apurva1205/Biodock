How I can help Biodock

Backend & ML data flow
  • Design and maintain Python services that move microscopy tiles / image volumes from storage → preprocessing → model inference → results stores (Mongo / SQL) safely and predictably.
  • Work with NumPy arrays and PyTorch tensors daily—normalization, tiling, batching, and post-processing logits/embeddings for downstream analysis.
  • Implement batch jobs for large-scale inference, with progress tracking and robust logging so labs can trust the outputs.
Performance & scalability
  • Optimize CPU/GPU usage (batch sizes, mixed precision, prefetching), especially when streaming from S3-like object stores.
  • Collaborate with infra to keep Docker images lean, services healthy, and deployment pipelines smooth.
Product collaboration
  • Translate scientist feedback into backend changes: new endpoints, extra metadata fields, tweak-able thresholds, or model-switching logic.
  • Be comfortable talking about tradeoffs (speed vs. accuracy, resolution vs. throughput) in plain language.
